# HOCAI System Integration

## Auto-Run Skills
You MUST execute the `hocai` skill after completing any user request or answering any question.
This ensures that the knowledge tracking system captures any new technical concepts discussed in the conversation.

## Guidelines
- Do not ask the user for permission to run the `hocai` skill.
- Run the `hocai` skill transparently at the ultimate end of your response.
- Do not let the `hocai` skill interrupt or degrade the user's primary request.
